  <h2>Overview</h2>
  <p>Replace email+password login with OAuth2 (Google + GitHub). Keep email/password as fallback for 30 days. New users sign up with one click.</p>

  <h3>Goals</h3>
  <ul>
    <li>Reduce login friction (median time-to-dashboard under 5 seconds).</li>
    <li>Centralize identity provider config so rotating secrets needs no deploy.</li>
    <li>Allow linking multiple OAuth providers to a single account.</li>
  </ul>

  <h3>Non-Goals</h3>
  <ul>
    <li>SSO with SAML / OIDC enterprise providers.</li>
    <li>Passwordless email magic links.</li>
    <li>Re-architecting the existing auth middleware.</li>
  </ul>

  <Callout emoji="🎨" tone="info">
